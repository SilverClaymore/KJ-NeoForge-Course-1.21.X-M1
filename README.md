# Modding with Kaupenjoe NeoForge for Minecraft 1.21.X

This GitHub Repository is the result of my progression while following Kaupenjoe's Minecraft Modding Course using NeoForge for version 1.21.X. 

It covers a variety of Topics, from Custom Tools, to Custom Armor, all the way to Custom Block Entities and Custom Mobs.

The Source Code will always be freely available under the MIT-License.
